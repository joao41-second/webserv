(http) Hypertext Transfer Protocol ?

e o protocol de comonicsao entre ciente e servidor.


HTTP request methods are actions that specifies operations to be performed
on the resource identified by the given Request-URI.
HTTP request methods used are : GET, POST, PUT, DELETE, HEAD, and OPTIONS.

O http tambem tem status codes com 200(OK), 404(Not Found), 500(Internal Server Error);

URI stands for Uniform Resource Identifier.
It is a string used to uniquely identify resources on the internet.


Tamnto o client como o server podem gerar Caching  para evitar fazer vairas vezes a mesma
chamada no casso do servidor chamadas ao programa de execusao.

headers -- sao utilizados para enviar metadados para o navegador e como intrepretar a resposta 
do server.
