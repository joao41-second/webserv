Site estático — Descarregar o próprio index

Este exemplo mostra como permitir que o utilizador faça download do ficheiro index.html da própria página.

Opções:

Descarregar este index Abrir index (mesma página)

Nota sobre compatibilidade

  • O atributo download é suportado pela maioria dos browsers modernos para URLs do mesmo domínio.
  • Se o servidor enviar Content-Disposition: attachment; filename="index.html", o browser irá forçar o download mesmo sem o atributo download.
  • Se estiver a testar localmente, use um servidor estático (por exemplo python3 -m http.server) em vez de abrir o ficheiro diretamente via file://.

Exemplo de comando para servir

python3 -m http.server 8000

Abra http://localhost:8000/ no seu browser e clique em "Descarregar este index".
